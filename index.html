Origen

El Problema vs La Solución

Diseñado por alguien que conoce el plantel, para el beneficio directo de compañeros y maestros.

El Problema

Comunicación lenta y confusa

La mayor pérdida de tiempo en simulacros y emergencias ocurre intentando explicar qué está pasando y en qué salón. Las alarmas auditivas no dan detalles exactos ni ubican el incidente.

La Solución

Flujo de datos crítico en tiempo real

Una plataforma digital que notifica en tiempo real lo qué está pasando y dónde exactamente — salón, sección y tipo de emergencia — a todos al mismo tiempo.

Funciones

El Omnipresencia de la app

Tres capas de alerta que garantizan que nadie se pierda una notificación crítica.

Alerta Visual Bloqueante

Cuadro de diálogo rojo con información de la emergencia que toma la pantalla completa, sin importar qué esté haciendo el usuario en su teléfono.

Notificaciones Push

El sistema avisa aunque el teléfono esté guardado o el usuario esté usando otra aplicación. Nada se pierde.

AlertListenerService

Un motor en segundo plano que nunca duerme, escuchando alertas de emergencia en tiempo real incluso cuando la app está cerrada.

Formulario Rápido

Selección de nivel (Secundaria o Preparatoria), tipo de incidente y área específica del accidente en segundos.

Confirmación en 2 Pasos

Pantalla de seguridad que muestra el mensaje exacto antes de enviar, para evitar alertas falsas y garantizar información precisa.

Simulacros Programados

Agenda y gestiona simulacros de emergencia con notificaciones anticipadas para que todo el plantel esté preparado.

Backend

Tecnología robusta

Stack moderno y probado que garantiza disponibilidad y seguridad en cada emergencia.

Firestore

Base de datos NoSQL en tiempo real de Firebase. Sincronización instantánea entre todos los dispositivos.

Firebase Auth

Acceso seguro exclusivo para el personal autorizado. Autenticación biométrica soportada.

Kotlin

Lenguaje de programación moderno, robusto y seguro. El estándar oficial de Android.

Jetpack Compose

Lo último en desarrollo Android. UI declarativa con Material3 de Google para una experiencia nativa fluida.

Android Studio

Desarrollada íntegramente en el IDE oficial de Android. Rápida, sencilla y poderosa.

Fácil Migración

Arquitectura lista para migrar de Firebase a SQL + PHP para integración con sistemas existentes del plantel.
